Toucher Couler

Présentation

Toucher Couler est une application mobile android que j'ai réalisé durant ma période d'initiation sur le framework android chez Dev-id. Pour info le framework android se base sur le langage Java. J'ai mis en oeuvre toute mes connaissances acquises lors de cette période d'initiation pour réaliser ce projet.

Le but du jeu est simple : un joueur place les bateaux sur chaque case et le deuxième joueur doit trouver les bateaux qui ont été placés. Seulement cinq bateaux maximum doivent être positionnés parmi les 25 cases de la grille. Le joueur peut placer le nombre de bateaux qu'il souhaite.Il doit ensuite choisir la difficulté (Facile, normal ou difficile).

Trouver les bateaux

Sur le deuxième écran, le joueur deux doit ensuite trouver les bateaux qui ont été placés par le joueur un avant que le nombre de tentative atteigne 0. Le nombre de tentative dépend du niveau de difficulté choisi par le joueur 1.
Une fois que tous les bateaux ont été trouvés, un message "GAGNER" s'affiche à l'écran ainsi qu'un bouton permettant de recommencer. Si les bateaux n'ont pas été trouver, c'est un message "PERDU" qui s'affiche.
